In-depth documentation on Reacon Trade is available at [docs.reacon.trade](https://docs.reacon.trade/).

# Local Development

The following assumes the use of `node@>=10`.

## Install Dependencies

`yarn`

## Compile Contracts

`yarn compile`

## Run Tests

`yarn test`
